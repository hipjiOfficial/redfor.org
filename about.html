About redfor.org

This project initially started with the goal of providing bombline players with easy access to the daily shop.

As time went on and development furthered, I started adding features to the project.

What I have now is redfor.org, a utility site for anything bombline related.

Redfor.org will receive continuous updates and maintenence, regardless of bombline's popularity.

Thank you for reading, and follow me on GitHub for more Hipji projects.
